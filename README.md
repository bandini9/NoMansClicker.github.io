**No Man's Clicker** is a browser-based idle exploration game inspired by the aesthetics of _No Man's Sky_ and the addictive mechanics of _Cookie Clicker_. Built entirely in a single HTML file with vanilla JavaScript and CSS.

## âœ¨ Features

- **Core Loop:** Mine planetary resources manually or automate the process with drones, probes, and extractors.
- **Research Lab:** Unlock powerful technology upgrades with witty flavor text.
- **Procedural Visuals:** CSS-only planetary rendering with dynamic atmosphere and orbiting units.
- **Expeditions:** Send your fleet on dangerous timed missions for massive rewards.
- **Galaxy Map:** Travel to different planets (Terra Nova, Magma Core, Toxic Prime, etc.) with unique production multipliers.
- **Save System:** Your progress is automatically saved to your browser's LocalStorage.
